## 프로젝트: 간단한 퍼즐 게임

### 1. 목표
- 브라우저에서 실행되는 카드 짝맞추기 퍼즐 게임을 만든다.
- 플레이어가 제한 시간 안에 모든 단계를 통과하도록 한다.
- 처음 시작 화면과 최종 종료 화면을 통해 게임 흐름을 분리한다.
- 실제 상용 시스템처럼 폴더 구성하고, html, css, javascript를 독립적인 파일로 만든다.

### 2. 대상 사용자
- 코딩 초보자
- 가볍게 즐길 수 있는 브라우저 게임을 원하는 사용자

### 3. 핵심 경험
- 게임을 열면 시작 화면이 먼저 보인다.
- 시작 버튼을 누르면 퍼즐이 시작된다.
- 각 단계는 난이도에 맞는 보드 크기로 진행된다.
- 모든 단계를 클리어하면 축하 종료 화면이 표시된다.
- 제한 시간을 넘기면 게임 오버가 된다.

### 4. 게임 흐름
1. 시작 화면 노출
2. 사용자가 `게임 시작` 버튼 클릭
3. 쉬움 단계부터 게임 시작
4. 퍼즐을 모두 맞추면 다음 단계로 자동 이동
5. 마지막 단계까지 완료하면 종료 화면 표시
6. 제한 시간 초과 시 게임 오버 처리

### 5. 단계 구성
- 쉬움: 3x3
- 중간: 4x4
- 중간1: 5x5
- 중간2: 6x6
- 어려움: 6x6

### 6. 제한 시간
- 각 단계마다 제한 시간을 둔다.
- 난이도가 올라갈수록 시간은 짧아진다.
- 시간이 0이 되면 입력을 막고 게임 오버 화면으로 전환한다.

### 7. 주요 기능
- 카드 짝맞추기
- 시도 횟수 카운트
- 카드 프리뷰 표시
- 단계 자동 진행
- 남은 시간 표시
- 시작 화면
- 종료 화면
- 다시 시작 기능

### 8. UI 구성
- 상단: 시도 횟수, 남은 시간, 단계 표시
- 중앙: 퍼즐 보드
- 하단: 안내 메시지
- 오버레이: 시작 화면, 종료 화면

### 9. 기술 스택
- HTML
- CSS
- JavaScript

### 10. 구현 범위
- 단일 `index.html` 중심으로 동작한다.
- 이미지나 외부 라이브러리 없이 구현한다.
- 모바일에서도 기본적으로 동작하도록 반응형 스타일을 사용한다.

### 11. 성공 기준
- 게임 시작 화면이 정상적으로 표시된다.
- 단계가 자동으로 진행된다.
- 제한 시간이 끝나면 게임 오버가 된다.
- 마지막 단계까지 완료하면 축하 종료 화면이 나온다.
